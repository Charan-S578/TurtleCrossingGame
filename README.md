## Turtle Crossing Game 🐢🚗

A fun arcade-style road crossing game built using Python’s Turtle graphics module. The player controls a turtle and must safely cross the road while avoiding moving cars. Each successful crossing increases the game level and car speed, making the game more challenging.



## Features
Player-controlled turtle movement
Randomly generated moving cars
Collision detection system
Level progression with increasing difficulty
Game Over screen
Smooth animation using Turtle graphics



## Technologies Used
Python 3
turtle module
random module
time module



## Controls

## Game Rules
Move the turtle to the top of the screen.
Avoid getting hit by moving cars.
Every successful crossing increases the level.
Car speed increases after each level.
The game ends when the turtle collides with a car.
